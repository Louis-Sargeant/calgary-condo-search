# Number One Calgary Lead Generation Goal

## Exact goal

Build the number one lead generation website in Calgary.

That is the goal for this project.

## What this means for the build

Every page, shortcode, CTA, form, IDX placement, and content decision must support lead generation.

The website should be built to capture serious Calgary condo buyers, condo sellers, building-alert leads, market-report leads, and valuation leads.

## Build priorities

1. Make it easy to search Calgary condos.
2. Make it easy to request condo alerts.
3. Make it easy to ask about a specific building.
4. Make it easy to request a condo value report.
5. Make every page push the visitor toward a useful action.
6. Keep the site Calgary-only.
7. Keep the Calgary phone number only: +1 (403) 800-6996.
8. Keep the approved myRealPage IDX intact.

## Page standard

Every lead-generation page should include:

- SEO title
- meta description
- H1
- useful intro copy
- IDX placement when relevant
- primary lead-capture CTA
- secondary CTA
- clear lead angle

## Do not do

- Do not use Vancouver contact details.
- Do not replace or scrape IDX data.
- Do not seed fake listings.
- Do not build pages without a lead-capture purpose.
- Do not make the site look generic.

## Operating rule

If a decision does not help make this the number one Calgary lead generation website, do not prioritize it.
