# AI First Homepage v2 Implementation

Final direction: AI first Calgary condo search homepage.

## Core layout

1. Header with logo, simple navigation, Calgary phone number.
2. Hero with Calgary skyline or tower background.
3. One large natural language request box.
4. Primary action: Ask AI.
5. Minimal secondary actions only:
   - Search Calgary Condos
   - Building Alerts
   - Condo Value Report
6. Shortcut cards below hero.
7. AI assistant explanation section.
8. IDX map and listings below the AI sections.

## Hero copy

Headline:
Tell us the condo you want. Let AI help narrow it down.

Subheadline:
Skip the dropdown overload. Describe your Calgary condo search in plain English and get matched with listings, buildings, alerts, and next steps.

Example request:
2 bedroom condo in Beltline under $500k with parking

## Visual direction

- White space first.
- Deep navy for authority.
- Gold for brand accent.
- Modern blue for AI interactions.
- Calgary skyline or Calgary Tower visual.
- No heavy form-first layout on homepage.
- No generic realtor template.

## Functional direction

The homepage should act like an AI intake layer before IDX.

Visitor can type a simple request. The system captures:
- area
- budget
- bedrooms
- must-haves
- timeline
- contact details

Phase one can use a normal form behind the AI request box. Phase two can add true conversational AI and voice.
