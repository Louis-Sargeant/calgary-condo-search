# Visual-First Lead Generation Design Rule

## Core direction

This Calgary condo lead-generation website must be visual, eye-catching, and easy to use. It should not feel word-heavy.

The site still needs strong SEO content, but the visitor experience should be built around visual sections, short copy, fast choices, strong calls to action, and clear lead paths.

## Design goal

Make the site feel like a modern Calgary condo search machine, not a long real estate article.

## Visual priorities

- Large hero sections with strong Calgary condo imagery.
- Card-based area grids.
- Price-range cards.
- Building-intent cards.
- Short CTA sections with bold buttons.
- Icons for buyer concerns: fees, parking, pets, views, building type, alerts, value reports.
- Visual comparison blocks instead of long paragraphs.
- Short bullets instead of dense copy.
- Strong spacing and clean section breaks.
- Mobile-first layout.

## Copy rules

- Keep on-page copy short and punchy.
- Put longer SEO content lower on the page or inside expandable sections.
- Lead with visuals and actions.
- Use headings that help people choose quickly.
- Every major section should have a clear next step.

## Page layout rule

Each major page should follow this visual order:

1. Strong hero with one sentence and two buttons.
2. Quick search or IDX section.
3. Visual cards for areas, price ranges, or buyer intent.
4. Trust/decision-help cards.
5. Alert form.
6. Seller/value report CTA.
7. Supporting SEO copy lower on the page.

## Lead-generation rule

Visual design must not just look good. It must drive action.

Every visual section should push visitors toward one of these actions:

- Search condos.
- Get condo alerts.
- Ask about a building.
- Request a condo value report.
- Read market updates.
- Compare communities.

## Brand direction

Use the Calgary Condo Search brand style:

- Deep blue: #0A1A2F
- Gold: #F0C75E
- Clean white space
- Strong contrast
- Modern cards
- Clear buttons
- Professional but not boring

## What to avoid

- Long walls of text near the top of pages.
- Generic blog-style layouts.
- Weak CTAs.
- Too many paragraphs before the visitor can search.
- Design that hides the IDX or forms.
- Pages that look like every other realtor site.

## Build standard

When a new page is planned, build it as a visual lead page first and an SEO page second.

The page must answer the visitor quickly, show them where to click, and make it easy to become a lead.
