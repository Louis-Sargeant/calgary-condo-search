# Upgraded Calgary Condo Hub Visual Test

## Purpose

Create a fast visual upgrade inspired by Realtor.com, but focused only on Calgary condos.

This should be tested on a draft WordPress page first, not on the working Calgary Condos page.

## Visual direction

The page should feel like a Calgary condo search engine, not a generic realtor website.

Top section should include:

- Large premium hero
- Search-first message
- Intent tabs
- Strong action buttons
- Shortcut cards
- Calgary Condo Assistant CTA
- Working myRealPage IDX below the visual block

## WordPress testing plan

1. Create a draft page called Visual Test - Calgary Condo Hub.
2. Use Full Width page template.
3. Add a Custom HTML block for the new hero and shortcut cards.
4. Add one Shortcode block below it.
5. Paste the proven working myRealPage shortcode.
6. Preview on desktop and mobile.
7. Only after approval, move the visual block to the live Calgary Condos page.
