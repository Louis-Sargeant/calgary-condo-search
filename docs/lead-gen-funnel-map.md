# Calgary Condo Search Lead Generation Funnel Map

## Goal

Turn the site into a Calgary condo lead-generation machine with multiple buyer, seller, investor, and building-research entry points.

## Primary conversion paths

### Buyer search path

Entry pages:
- Calgary Condos For Sale
- Price-range pages
- Area pages
- IDX listing result pages
- Listing detail pages

Primary CTA:
- Search live Calgary condo listings.
- Get Calgary condo alerts.

Lead capture:
- Condo alert form.
- Showing/contact request through IDX.
- Building alert request.

Follow-up intent:
- Buyer wants listings, building guidance, or help narrowing options.

### Building research path

Entry pages:
- Calgary Condo Buildings
- Individual future building pages
- Area pages

Primary CTA:
- Watch a Calgary condo building.
- Get building alerts.

Lead capture:
- Building alert form.
- Condo alert form with building name in message.

Follow-up intent:
- Buyer is interested in a specific building or area.

### Seller value path

Entry pages:
- Calgary Condo Value Report
- Building pages
- Market report page

Primary CTA:
- Request a Calgary condo value report.

Lead capture:
- Value report form.

Follow-up intent:
- Owner may sell now or later and wants a condo-specific pricing plan.

### Investor path

Entry pages:
- Calgary Condo Investment Search
- Price pages
- Area pages

Primary CTA:
- Request investor condo guidance.

Lead capture:
- Investor guidance form.

Follow-up intent:
- Buyer wants help with due diligence, fees, building fit, and exit strategy.

## Page CTA rules

Every page should include at least one of these:

- Search listings.
- Get condo alerts.
- Set building alerts.
- Request condo value report.
- Call Calgary number: +1 (403) 800-6996.

## Lead quality questions

For buyer/alert forms:
- Name.
- Email.
- Phone.
- Preferred Calgary area.
- Budget.
- Timeline.
- Building preferences or must-haves.

For value report forms:
- Name.
- Email.
- Phone.
- Building/address.
- Unit type.
- Approximate size.
- Parking/storage.
- Timeline.
- Selling goal.

For investor forms:
- Name.
- Email.
- Phone.
- Budget.
- Preferred area.
- Cash flow vs appreciation priority.
- Timeline.
- Financing/pre-approval status.

## Compliance and safety rules

- Do not create fake active listing content.
- Do not represent sold data unless verified and allowed.
- Do not change myRealPage IDX plugin code.
- Keep brokerage and IDX compliance visible where required.
- Use Calgary-only site contact info.
