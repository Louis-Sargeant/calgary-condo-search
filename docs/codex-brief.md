The WordPress site already has a working myRealPage IDX search page. Do not replace the IDX plugin and do not delete the IDX shortcode.

## Goal

Create a lightweight WordPress plugin that adds conversion-focused sections and styling around the existing IDX pages.

1. Plugin scaffold.
2. Shortcode for a hero section.
3. Shortcode for a condo alert lead form section.
4. Shortcode for trust/value proposition cards.

6. Clear install instructions.

## Rules

- Keep Coming Soon mode on until approval.
