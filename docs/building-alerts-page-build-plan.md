# Building Alerts Page Build Plan

## Page

Building Alerts

URL:

```text
/building-alerts/
```

## Page goal

Capture Calgary condo buyers who want alerts for a specific building, area, or condo type.

## H1

```text
Watch a Calgary Condo Building
```

## SEO title

```text
Calgary Condo Building Alerts | Calgary Condo Search
```

## Meta description

```text
Watch specific Calgary condo buildings and get alerts when new listings match your preferred building, budget, and timing.
```

## Core message

If a buyer already knows the building, area, or lifestyle they want, they should not depend on random searching. Building alerts help them track the right opportunities faster.

```text
[ccl_hero primary_text="Set Up Building Alerts" primary_url="#condo-alerts" secondary_text="Search Calgary Condos" secondary_url="/calgary-condos/"]
[ccl_building_cta]

```

## Lead form focus

Use the form message field to ask for:

- Building name
- Preferred Calgary area
- Budget
- Parking needs
- Pet rules
- Timeline

## CTA rules

- Primary CTA: Set Up Building Alerts
- Secondary CTA: Search Calgary Condos
- Keep copy focused on Calgary condo buyers
- Do not add fake building inventory
