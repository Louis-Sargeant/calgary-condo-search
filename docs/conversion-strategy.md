# Conversion Strategy — Calgary Condo Search

## Goal

Build a Calgary condo lead-generation site that turns search traffic into real buyer and seller conversations.

## Primary visitor types

1. Active condo buyers searching live listings.
2. Early-stage buyers comparing buildings and areas.
3. Building-specific buyers waiting for the right unit.
4. Calgary condo owners checking value before selling.
5. Investors watching price ranges and building types.

## Main conversion paths

### Buyer path

Search listings → compare area/building → request condo alerts → follow-up conversation.

Primary CTAs:

- Search Calgary Condos
- Get Condo Alerts
- Set Up Building Alerts
- Ask About This Building

### Seller path

Read market/building/value content → request condo value report → follow-up conversation.

Primary CTAs:

- Request Condo Value Report
- Call Calgary Number
- Ask About Your Building

### Authority path

Read market report or building guide → subscribe to updates → future buyer/seller lead.

Primary CTAs:

- Get Market Updates
- Watch This Building
- Get Condo Alerts

## Page-level conversion rules

- Every page needs one primary CTA above the fold.
- Every page needs one secondary CTA below the main content.
- Buyer pages should not push too many seller CTAs early.
- Seller pages should not bury the value-report CTA.
- IDX search should remain central on search pages.
- Lead forms should stay short enough for mobile users.
- The Calgary phone number should be visible on high-intent sections.

## Trust signals

Use subtle trust signals without making the site feel gimmicky:

- Calgary-focused condo guidance.
- Live IDX search.
- Building and fee awareness.
- Straight advice before showings.
- Real Broker brokerage reference.
- Always In Your Corner.

## What to avoid

- Fake listings.
- Unsupported market claims.
- Too many slogans.
- Vancouver contact information.
- Generic real estate copy that could fit any city.
- Long forms before the visitor sees value.
- Launching before IDX and forms are verified.
