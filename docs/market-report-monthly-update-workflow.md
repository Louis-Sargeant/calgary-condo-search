# Market Report Monthly Update Workflow

## Rule

The Calgary Condo Market Report should be updated monthly from CREB® data.

Use CREB® as the primary source because it is the official Calgary Real Estate Board market data source.

Do not use Google AI summaries as the source.

Do not use competitor wording.

## Source

Primary source:

```text
CREB® Housing Statistics
https://www.creb.com/Housing_Statistics/
```

Monthly source type:

```text
City of Calgary Monthly Statistics Package
```

## Timing

Check for the new CREB monthly package at the start of each month after CREB releases the prior month's report.

Example:

- Early June: use May report
- Early July: use June report
- Early August: use July report

## Data to update each month

For apartment-style condos, update:

1. Sales
2. New listings
3. Inventory
4. Months of supply
5. Benchmark price
6. Sales-to-new-listings ratio
7. Year-over-year benchmark price change
8. CREB's apartment/condo commentary summary

## Page sections to refresh monthly

Update these sections on the Calgary Condo Market Report page:

- Snapshot cards
- Buyer interpretation
- Seller interpretation
- Market direction language
- Source/date line

## Interpretation rules

Use plain-English guidance:

- What the data means for buyers
- What the data means for sellers
- What it means for building selection
- What buyers should compare before booking showings
- What sellers need to compete against active inventory

Avoid unsupported claims like:

- The market is crashing
- Prices will go up
- Prices will go down
- Buyers can lowball every condo
- Sellers cannot sell

Keep the tone direct, factual, and useful.

## CTA rules

Every market report update should keep these CTAs:

- Search Calgary Condos
- Get Calgary Condo Alerts
- Get My Condo Value Report

## Build note

This monthly report should strengthen trust and lead capture. It should not become a generic blog post. It should be a monthly Calgary condo decision tool.
