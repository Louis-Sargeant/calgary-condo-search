# Calgary Condo Search Supporting Page Content

Use these page blocks to build approved placeholder or live supporting pages in WordPress. Keep the existing myRealPage IDX plugin and IDX data unchanged.

## Global page rules

- Use Calgary-only contact information.
- Do not add fake MLS listings, fake market statistics, scraped listing data, or placeholder property cards.
- Use the existing IDX shortcode/embed only where approved.
- Keep Coming Soon mode on until launch approval.
- Every supporting page should route visitors to `#condo-alerts`, `/calgary-condos/`, or `/condo-value-report/`.
