# Current Calgary Condo Lead-Gen Page Stack

Recommended page order for the main Calgary condo search landing page:

```text
[ccl_hero primary_url="#idx-search" secondary_url="#condo-alerts"]
[ccl_quick_search]

[ccl_idx_shell]
    Place the approved IDX shortcode here.
[/ccl_idx_shell]

[ccl_action_router]
[ccl_market_snapshot]
[ccl_building_scorecard]
[ccl_building_checklist]
[ccl_trust_strip]
[ccl_lead_magnet]
[ccl_faq]
[ccl_alert_form]
[ccl_sticky_cta]
[ccl_site_footer]
```

## Conversion logic

1. Hero gets the visitor moving.
2. Quick search gives a fast entry point.
3. IDX search handles live inventory.
4. Action router prevents dead-end browsing.
5. Market and building sections educate the buyer.
6. Trust strip supports credibility before form action.
7. Lead magnet captures quieter research-stage visitors.
8. FAQ removes objections.
9. Alert form converts.
10. Sticky CTA keeps the conversion path visible.
